A simple weather application built using **JavaScript, HTML, and CSS**, and by the **OpenWeather API**.  
It allows users to search for any city and instantly view the current temperature, humidity, wind speed, and weather conditions with dynamic icons.

---

## Features
- Search for weather by city name
- Displays temperature, humidity, and wind speed
- Weather icons change dynamically based on conditions (Clear, Clouds, Rain, Snow, etc.)


---

## Setup Instructions

1. **Clone the repository**
2. signup in openweather site
3. get api key and paste it in index.js where is is told
